Detailed Description

System clock custom kernel implementation.

sysclk_source_tick reschedules tasklet source_tasklet that is represented by function source_tasklet_callback. source_tasklet_callback function should be called synchronously, to call synchronously msec_scheduler. For this purpose TDM is good aim. If source_tasklet_callback wasn't called from any synchronous source, sys_clock_fail will be called.

static void sys_clk_ctl_tmr_callback ( u_long  arg)
static

Timer callback function. Timer is created and configured in sys_clock_init. Function changes global variable curr_clock.

Parameters
arg: Is not used.
  1. Find first non-NULL sysclk_source structure in linked list source_list.
  2. Define how many ticks passed after last call.
  3. If source_tasklet_callback wasn't called, set failure.
  4. If source_tasklet_callback was called, change curr_clock.
  5. If curr_clock exceeds limits that is set by sysclk_source.s_sysclk_config.normal_clock_low and sysclk_source.s_sysclk_config.normal_clock_high - call sys_clock_fail.
  6. If not, call sys_clock_normal.
  7. Reset sys_clock_first_time flag.
  8. Reset was_gpt_int flag.
  9. Change the expiration time of an already scheduled sys_clk_ctl_tmr.
