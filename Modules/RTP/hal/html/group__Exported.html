int msec_sched_add_task ( void(*)(unsigned long)  funct,
unsigned long  data,
uint  period 
)

Adds new function to execute in millisecond scheduler.

Parameters
funct: Function that represents new task.
data: Argument that will be passed to this function.
period: Execution period. Will be used to restore msec_task.counter in function msec_scheduler.
  1. Allocate new msec_task structure.
  2. Fill msec_task structure's fields.
  3. Create tasklet inside msec_task structure with given function and its parameter.
  4. Disable created tasklet.
  5. Add new msec_task structure to the #msec_queue list.
  6. Enable created tasklet. To start tasklet, function 'tasklet_schedule' called in msec_scheduler.

int sysclk_source_add ( struct s_sysclk_config *  cfg)

This function initializes and adds new sysclk_source to the source_list linked list.

Parameters
cfg: Configuration for the sysclk_source.
Returns
System clock source given ID.
  1. Take source_list_lock semaphore.
  2. Allocate new sysclk_source structure.
  3. Set sysclk_source structure's ID from global variable sysclk_source_id.
  4. Add new new sysclk_source structure to the source_list.
  5. Return source_list_lock semaphore.

int sysclk_source_tick ( int  id,
unsigned long  tick_count 
)

Main purpose of this function is to call for 'tasklet_schedule' to reschedule source_tasklet. This function changes tick count to the #sysclock_source structure with the given ID. This function represented to kernel for use in other modules.

Parameters
id: ID of #sysclock_source to operate on. ID was returned from sysclk_source_add.
tick_count: New tick count to set.

If #sysclock_source for given ID is the first in the list, it schedules source_tasklet.
