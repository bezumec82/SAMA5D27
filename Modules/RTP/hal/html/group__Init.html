void msec_sched_deinit ( void *  context)

Millisecond scheduler de-initialization routine.

Parameters
context- Not used in this routine. Structure hal_context will be passed by init_startup.
  1. Take msec_queue_lock semaphore.
  2. Iterate over a list #msec_queue.
  3. Get msec_task structure from the found list entry.
  4. Disable and kill tasklet inside found msec_task structure.
  5. Delite found list entry.
  6. Deallocate found msec_task structure.
  7. Return msec_queue_lock semaphore.
